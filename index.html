Privacy Policy for Tutor Polly

Effective Date: Dec15, 2023

1. Introduction

Welcome to Tutor Polly! This Privacy Policy explains how we collect, use, and disclose information when you use our iPhone application. By using our app, you agree to the terms outlined in this policy.

2. Information We Collect

2.1 User Input Data

Our app uses the ChatGPT API to correct language sentences. Any text input provided by users for correction is processed by the API in real-time. We do not store or transmit this user input data to our servers.

2.2 Speech Practice Data

For speech practice, we utilize Google Text-to-Speech. The text provided by users for speech practice is sent to Google's servers for real-time speech synthesis. We do not store or retain this speech practice data.

2.3 Analytic Data

We use Apple's built-in analytics to collect anonymized and aggregated usage information. This may include device information, app crashes, and general usage statistics. This data is collected by Apple and is subject to Apple's privacy policy.

3. How We Use Information

3.1 User Input Data

User input data sent to the ChatGPT API is processed in real-time for language sentence corrections within the app. We do not store or use this data for any purpose beyond the immediate correction process.

3.2 Speech Practice Data

Text provided for speech practice is sent to Google's servers for real-time speech synthesis. We do not store or use this data beyond the immediate synthesis process.

3.3 Analytic Data

Analytic data collected by Apple is used to understand user behavior, improve app performance, and enhance user experience. This information is only accessed in aggregate form, and individual user activities are not tracked.

4. Information Sharing and Disclosure

We do not sell, trade, or otherwise transfer user input or speech practice data to third parties. Analytic data collected by Apple is subject to Apple's privacy policy.

5. Security

We take reasonable measures to protect user input and speech practice data. However, it's important to note that no method of transmission over the internet or electronic storage is 100% secure.

6. Changes to this Privacy Policy

We may update this Privacy Policy from time to time. Any changes will be reflected in the app, and your continued use of the app after the effective date of the updated policy constitutes your acceptance of the changes.
